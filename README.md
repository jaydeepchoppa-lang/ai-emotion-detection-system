# AI Emotion Detection System

## 📌 Project Overview

This project is an AI-based Emotion Detection System that analyzes facial expressions using a webcam and detects emotions such as Happy, Sad, Angry, etc.

## 🚀 Features

* Real-time emotion detection
* Uses Deep Learning (DeepFace / OpenCV)
* Stores emotion logs in CSV file
* Simple and user-friendly interface

## 🛠️ Technologies Used

* Python
* OpenCV
* DeepFace
* FastAPI (if used)

## ▶️ How to Run

1. Install dependencies:
   pip install -r requirements.txt

2. Run the application:
   python app.py

## 📁 Files Included

* app.py → Main application
* requirements.txt → Dependencies
* emotion_log.csv → Stored emotion data
